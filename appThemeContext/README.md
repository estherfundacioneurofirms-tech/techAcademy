Crea una pequeña aplicación de React que permita cambiar entre modo claro-oscuro usando useContext.
La aplicación tendrá varios componentes, pero el estado del tema no se pasará por props. En su lugar, se compartirá mediante un contexto global llamado ThemeContext.

La app debe tener como mínimo:
App:
	-Header
	-MainContent
	-ThemeButton
El button deberá cambiar entre el modo claro y oscuro.
Cuando el tema cambie, también deberán cambiar los colores de la aplicación
